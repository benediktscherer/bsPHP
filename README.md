# bsPHP

## required
* php7.2
* node 9.9.0

### init
```txt
Creating App
    $ cp example.env .env
    $ composer install
    $ npm install
    $ grunt
```

### develop

```txt
    $ npm run dev
    $ grunt <command>

  Commands:

    default     compile all files to dist/
    patternlab  generate patternlab
    dev         compile all files to dist/, create source-maps, start watcher
```
